    keyframes

    @mixin keyframes($animationName) { ... }

    Description

    This is the main mixin for creating keyframe animations with cross-browser compatibility automatically set up. It needs the name and also the body with the keyframe locations and properties.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $animationName

    The name of the animation to be referenced

    		String—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    @include keyframes(name-here) {
   0%   { color: black; }
   50%   { color: white; }
   100% { color: black; }
}

@include animation('name-here 1s linear 0s infinite');

    fix-flex

    @mixin fix-flex($fix-direction, $fix-width, $fix-margin) { ... }

    Description

    The fix-cell and flex-cell mixins included components of the fix-flex mixin, which is used for items with a fixed width on one side and flexible on the other.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $fix-direction

    If the fixed element is on the left or right side

    		String—none
    $fix-width

    The width of the fixed element

    		Length—none
    $fix-margin

    The margin length between the two items

    		Length—none

    Example

    @include fix-flex('left', 2em, 10px);

    Requires

    • [mixin] fix-cell
    • [mixin] flex-cell

    Author

    • Krispin Dolbear, EMA

    modular-size-map

    $modular-size-map: (
   h1: $base-font-scale + 6,
   h2: $base-font-scale + 5,
   h3: $base-font-scale + 4,
   h4: $base-font-scale + 3,
   h5: $base-font-scale + 2,
   h6: $base-font-scale + 1,
   p: $base-font-scale,
);

    Description

    This uses the Bourbon modular-scale function to create the structure of how fonts will be sized in the modular scale.

    Type

    Map

    See

    • [variable] base-font-scale
    • [variable] modular-scale-ratio

    Links

    • http://bourbon.io/docs/#modular-scale

    base-font-family

    $base-font-family: $helvetica !important;

    Description

    The default font family for all text except headers. Can be a specific font family or a Bourbon font variable.

    Type

    Variable or String(s)

    heading-font-family

    $heading-font-family: $base-font-family;

    Description

    The default font family for only headers. Can be a specific font family or a Bourbon font variable. By default, it's the same as the base font.

    Type

    Variable or String(s)

    base-font-size

    $base-font-size: 1em;

    Description

    The font size that all other element font sizes are scaled off of. This is often the size of basic paragraph text, and is scaled up for headers.

    Type

    Unit of size(em)

    See

    • [variable] modular-scale-ratio

    base-font-scale

    $base-font-scale: 0;

    Description

    This is the initial scale of the modular scale ration you want your font sizes to begin at. Typically left at zero, which will make your basic text the same of your base font size. Header font sizes will scale up.

    Type

    Integer

    See

    • [variable] base-font-size
    • [variable] modular-scale-ratio
    • [variable] modular-size-map

    modular-scale-ratio

    $modular-scale-ratio: $major-third;

    Description

    This is the modular scale the fonts will use for sizes. It can be a Bourbon variable or an integer with decimals.

    Type

    Integer

    See

    • [variable] modular-size-map
